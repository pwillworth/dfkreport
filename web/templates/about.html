DeFi Kingdoms: Lila's Ledger

About Lila's Transaction History Service

Lila's Ledger was created in late 2021 as part of the first Defi Kingdoms code bounty for a Transaction History Report Generator. After winning the bounty, Lila's Ledger continues to be updated to evolve along side the ever changing Defi-Kingdoms ecosystem.
